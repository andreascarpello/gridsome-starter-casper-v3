# About the starter

Gridsome starter that is ideal for portfolios and blogs that uses markdown for the content and tailwindcss for the styles.

This starter is based on [Ghost Casper V3](https://github.com/TryGhost/Casper).

## Key features

* Built with tailwindcss
* Responsive 
* Blog
* Author pages
* Tags
* Categories
* Dynamic custom pages
* Pagination

## Installation

Run the following command to create a new gridsome project 

```
gridsome create my-blog https://github.com/jammeryhq/gridsome-starter-casper-v3
```
